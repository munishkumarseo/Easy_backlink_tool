FAQs

How this SEO prompt tool works

A quick reference for the upload flow, keyword matching, Google Sheet loading, AI output, and backlink content workflow.

Upload or load a public sheet Select keyword and target URL Generate, edit, and copy output
What does this tool do?

It reads a keyword sheet, matches each keyword with its target URL, generates editable prompts, and sends those prompts to the AI generator for SMO posts, blogs, micro blogs, ads, and image prompt packages.

Can it load Google Sheets?

Yes. Paste a public Google Sheet URL and the tool fetches the sheet fresh, parses keywords and target links, and supports complex product-link sheets with multiple variants under one keyword.

Why does the Target URL dropdown change?

If a keyword has multiple product links or variants, the dropdown shows those URL options. If only one link is available, it behaves like a simple target URL field with copy and preview controls.

What content types are supported?

The tool currently supports SMO, blog, micro blog, ad, and image prompt formats. Each type uses a different prompt template and tone rules for backlink-friendly content generation.

Does it write promotional content?

Non-ad prompt types are designed to stay informational and topic-first. The prompts and backend guardrails restrict sales language, unsupported claims, medical claims, and first-person brand promotion.

Can I copy formatted AI output?

Yes. Use Rich Copy to copy formatted output from the editor, or Plain Text to simplify it. The generated prompt itself can also be copied and edited before generation.
